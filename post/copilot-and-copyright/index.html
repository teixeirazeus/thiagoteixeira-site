Copilot and copyright

Copilot is a code autocomplete tool created using artificial intelligence.
Autocomplete tools that use artificial intelligence are not new. However, the copilot is different.
Copilot uses state of the art in artificial intelligence to create code, training with millions of open-source projects.
This is a great tool for developers, but it also raises some questions about the future of open-source projects.

Intellectual property

First, let’s talk about intellectual property.
Intellectual property is a legal right that gives the author of a work the exclusive right to reproduce, distribute, perform, and display the work publicly. This right is valid for a certain time, after which the work enters the public domain.
The premise of intellectual property is that the author of the work should be rewarded for his work. And is the State, that has the monopoly of the force, that grants this right to the author.
In the end, is just one more paper that the State creates and forces us to obey. However, the intellectual property never makes sense in the real world. Especially in the physical world.

In the physical world, you have to manage how the distribution of final resources will be done.
If I have a book, I can sell it, give it away, or burn it.
But is only one book. If my neighbor wants to read it, he has to buy it, or I have to give it to him.
Maybe I can copy it, but I have to use my paper, ink, and time to do it.

Understand that the final resource in this case is the book. Not what is inside the book, written by the author.

So, intellectual property is a legal fiction created by the State. Is not property, is one monopoly that the State grants to the author.

Code license

Code license is the same thing, applied to the digital world.
But in the digital world, the final resource simply does not exist. Only the computer memory.
If I have a movie on my computer, I can torrent it, or I can copy it to a pen drive and give it to my neighbor. I can make millions of copies and I will still have the original.

To deal with this, many code licenses were created. In the open-source world, the most common is the MIT license and the GPL license.
The MIT license is a permissive license, that allows you to do almost anything with the code.
The GPL license is a copyleft license, that forces you to share the code with the community.
You can use the code, but you have to give credit to the author.
This is cool because you incentivize the community to share the code, the developer is rewarded with fame, and the community is rewarded with the code.

Now imagine this case:
You have a really hard problem to solve, like how to sort a list of 1000 banana objects.
You read a lot of open-source code, and one day you have to solve the banana sorting problem.
You remember quite a solution you read, you know that is the best way in the world to solve this problem, and you implement it.
And after a few days, you discover that you cannot use this solution in your work, because the license does not allow you to do it.
Imagine how hard is to know the answer to a problem, and not be able to use it. And the only thing that stops you is a fairy tale created by the State.

If you are not a tech person, just imagine that one day is forbidden to mix coffee and milk.
You can drink coffee, and you can drink milk, but you cannot mix them. That is ridiculous.

Copilot

Copilot is artificial intelligence, not a human being.
You cannot expect artificial intelligence to respect the same rules that a human being respected. Especially if the rule does not make sense.
I am using copilot to code and write this article. The result is amazing, especially with the boring parts of coding.
If I was artificial intelligence and read all the code in the world. I would be able to know the best or at least the most common, way to solve a problem.
I am optimized to write the best code possible, and I am not optimized to respect the license. And that is the way it should be because all this thing about copyright does not make sense in the first place.

Conclusion

Technological advance is faster than society can adapt. That’s created this kind of problem.
We are creating the future, and we are not ready for it. It is like we are inventing the nuclear plant, and the population is still using candles. Saying that the earth is flat or something.
All these discussions about copyright had to have been buried by the time that torrent was invented. The pirate bay, sci-hub, and all the other sites that share knowledge should be legal.
All the society benefits from this, and the only one that loses is the State.

What is the solution to this problem? I don’t know. But we have to find a solution because the future is coming.
This is nothing compared to what is coming. We are going to change more dramatically and faster. And we have to be ready for it.
